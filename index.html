Getting Started

Read the documentation

The UI Kit requires that you use a single .less file to add to your .cshtml and .less files. By adding it to your .cshtml files, you will get access to drop-in classes that will apply styles to common elements such as buttons, as well as access to a responsive grid system. By importing it in your .less files, you'll get access to commonly used variables related to colors and type.

Add this to your .cshtml files before your custom styles

<link rel="stylesheet/less" type="text/css" href="lib/base.less">
<script src="http://cdnjs.cloudflare.com/ajax/libs/less.js/1.7.0/less.min.js"></script>
<script>less.watch();</script>

Add this line to the top of your .less files

@import "lib/base.less";

Typography

Introduction

Typography is an important design element for Stack Overflow Careers. Strong typography applied consistently can not only build brand recognition and trust, but it can also help establish reliable communication patterns.

Typefaces

To communicate a professional, yet friendly personality, Stack Overflow Careers has selected the typeface Merriweather Sans ST. This font was designed specifically for on-screen use, allowing for greater control of line-heights and letter-spacing natively within our CSS without resorting to Javascript.

While we may use other weights when the situation arises, most copy will one of Light, Normal, and Bold styles.

@fontSize-xxsmall:  @fontSize-base * 0.714285714;
@fontSize-xsmall:   @fontSize-base * 0.857142857;
@fontSize-small:    @fontSize-base * 0.928571429;
@fontSize-base:     14;
@fontSize-medium:   @fontSize-base * 1.285714286;
@fontSize-large:    @fontSize-base * 1.714285714;
@fontSize-xlarge:   @fontSize-base * 2.571428571;
@fontSize-xxlarge:  @fontSize-base * 3.428571429;
@fontSize-xxxlarge: @fontSize-base * 5.142857143;

//  Font Size Mixin Guard
.fontSize (@s) {
  & when (@s = xxs)    { .fontRem(ceil(@fontWeight-xxsmall)); }
  & when (@s = xs)     { .fontRem(ceil(@fontWeight-xsmall)); }
  & when (@s = s)      { .fontRem(ceil(@fontWeight-small)); }
  & when (@s = base)   { .fontRem(ceil(@fontWeight-base)); }
  & when (@s = m)      { .fontRem(ceil(@fontWeight-medium)); }
  & when (@s = l)      { .fontRem(ceil(@fontWeight-large)); }
  & when (@s = xlarge) { .fontRem(ceil(@fontWeight-xlarge)); }
  & when (@s = xxl)    { .fontRem(ceil(@fontWeight-xxlarge)); }
  & when (@s = xxxl)   { .fontRem(ceil(@fontWeight-xxxlarge)); }
}

//  Font REM Sizing Mixin
.fontRem (@fs) {
  @remValue: @fs / @fontSize-base;
  font-size: ~"{fs}px";
  font-size: ~"{remValue}px";
}

// Apply the mixin
h1 { .fontSize(l); }

// This will output:
h1 { 
  font-size: 36px;
  font-size: 2.571428571rem;
}

Font Weight

With the addition of webfonts, the use of font-weight is slightly different. Previously a "bold" weight was programmatically determined using an algorithm to fatten a font. Webfonts however are hinted specifically for declared weight values.

  • Use a numerical value to declare a font-weight.
  • Avoid entering raw font-weight numbers directly into LESS files. Instead use one of the font-weight variables.
//  Font Weight Variables
@fontWeight-light:     200;
@fontWeight-normal:    400;
@fontWeight-bold:      700;

//  Font Weight Mixin Guard
.fontWeight (@w) {
  & when (@w = 2) { font-weight: @fontWeight-light; }
  & when (@w = 4) { font-weight: @fontWeight-normal; }
  & when (@w = 7) { font-weight: @fontWeight-bold; }
}

// Apply the mixin
h1 { .fontWeight(2); }

// This will output:
h1 { font-weight: 100; }

Line Heights

  • Do not declare values with line-heights (i.e. px, em). Keeping them unitless allows the value to be relative to the declared font-size, which provides greater flexibility.
  • Instead of trying to rememeber a rather long number like 1.428571429 or variations thereof, use the pre-defined variables that have been created.
  • Please avoid entering raw numbers within the LESS.
//  Line Height Variables
@lineHeight-tightest:       (@fontSize-base + 2) / @fontSize-base;
@lineHeight-tighter:        (@fontSize-base + 4) / @fontSize-base;
@lineHeight-tight:          (@fontSize-base + 6) / @fontSize-base;
@lineHeight-base:           @fontSize-base / @fontSize-base;
@lineHeight-loose:          (@fontSize-base + 8) / @fontSize-base;
@lineHeight-looser:         (@fontSize-base + 10) / @fontSize-base;
@lineHeight-loosest:        (@fontSize-base / @fontSize-base) * 2;

//  Line Height Mixin Guard
.lineHeight (@lh) {
  & when (@lh = base)     { line-height: @lineHeight-base; }
  & when (@lh = tight)    { line-height: @lineHeight-tight; }
  & when (@lh = tighter)  { line-height: @lineHeight-tighter; }
  & when (@lh = tightest) { line-height: @lineHeight-tightest; }
  & when (@lh = loose)    { line-height: @lineHeight-loose; }
  & when (@lh = looser)   { line-height: @lineHeight-looser; }
  & when (@lh = loosest)  { line-height: @lineHeight-loosest; }
}

// Apply the mixin
h1 { .lineHeight(loose); }

// This will output:
h1 { line-height: 1.571428571; }

Letter Spacing

Character tracking previously available only through Javascript libraries such as Kerning.js and now available to us within our CSS because of Merriweather's on-screen optimizations.

  • To maintain consistency with letter-spacing application, avoid entering raw values within the LESS. Instead use one of the letter-spacing variables.
  • If a raw number must absolutely be entered within the LESS, please comment it appropriately so others can understand why.
//  Letter Spacing Variables
@letterSpacing-zero:       0;
@letterSpacing-base:       -0.5px;
@letterSpacing-tight:      @letterSpacing-base * 2;
@letterSpacing-tighter:    @letterSpacing-base * 5;
@letterSpacing-tightest:   @letterSpacing-base * 10;

//  Letter Spacing Mixin Guard
.letterSpacing (@ls) {
  & when (@ls = 0)        { letter-spacing: @letterSpacing-zero; }
  & when (@ls = base)     { letter-spacing: @letterSpacing-base; }
  & when (@ls = tight)    { letter-spacing: @letterSpacing-tight; }
  & when (@ls = tighter)  { letter-spacing: @letterSpacing-tighter; }
  & when (@ls = tightest) { letter-spacing: @letterSpacing-tightest; }
}

// Apply the Mixin
h1 { .letterSpacing(tighter); }

// This will output:
h1 { letter-spacing: -2.5px; }
