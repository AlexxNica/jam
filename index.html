Getting Started

An overview of the Stack Overflow Careers UI Library, how to use it, formatting and class conventions, examples, and more.

Overview

The UI Kit gives developers access to classes and variables to style their pages with the opinion of the design team baked in. There are a number of advantages that using the UI Kit gives the developers, designers, and other teams related to Careers:

  • It should cut down the amount of code that developers need to write, relying on the library for standard elements so he or she can focus on page specific design
  • It gives designers the ability to standardize the look and feel of Careers and propagate changes easily throughout the site
  • The documentation gives marketing, sales, and other teams that need design guidance for Careers a place to look at without having to consult the designers directly

Installation

The UI Kit requires that you use a single .less file to add to your .cshtml and .less files. By adding it to your .cshtml files, you will get access to drop-in classes that will apply styles to common elements such as buttons, as well as access to a responsive grid system. By importing it in your .less files, you'll get access to commonly used variables related to colors and type.

Add this to your .cshtml files before your custom styles:

<link rel="stylesheet/less" type="text/css" href="less/base.less">

<script src="http://cdnjs.cloudflare.com/ajax/libs/less.js/1.7.0/less.min.js"></script>
<script>less.watch();</script>

Add this line to the top of your .less files:

@import "lib/base.less";
